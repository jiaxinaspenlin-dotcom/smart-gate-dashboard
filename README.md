# Smart Gate Dashboard (Airport Demo)

Mobile-style airport dashboard prototype inspired by a “SMART gate” concept for improving passenger flow and airport experience.

**Tech:** React + Vite + Leaflet (maps) + Recharts (charts)  
**Status:** Prototype demo using sample estimates (not an official TSA/Airline feed)

## Features
- **Dashboard:** airport map + flight summary + TSA wait-time trend chart (demo data)
- **Itineraries:** sample trip list UI
- **TSAInfo:** checkpoint wait-time list (demo data)
- **Directions:** simple step-by-step directions + quick link to Google Maps

## Data Notes (Important)
This project uses **sample/demo values** to simulate TSA wait times and flight info.  
Not affiliated with TSA, American Airlines, or any airport authority.
